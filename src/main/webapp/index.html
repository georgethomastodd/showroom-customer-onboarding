Local installation of the showcase

You can run this showcase on a local Camunda BPM installation (as an alternative to use the showroom). Follow these steps to get it running.
  • Download and unzip latest Camunda BPM EE WildFly 10 Distribution from the enterprise download page. Hint: The deployment folder is now located in BASE_PATH/camunda-bpm-ee-wildfly10-7.XXX-ee\server\wildfly-XXX.Final\standalone\deployments, whereas XXX might differ depending on the concrete version downloaded.
  • Add the Process Application containing the showcase in the deployment folder: insurance-application.war.
  • Delete the existing Camunda Webapp in the deployment folder (named camunda-webapp-ee-jboss-7.XXX-ee.war) and add the customized version to the deployment folder instead: camunda-webapp-customized.war.
